HelioWeb Documentation

Work-to-Author Links

Work-to-author links have been created only for authors with ORCiDs recorded by ADS. They are sorted by author display-name and are qualified by whether the ORCiD was reported to ADS by a publisher (pub), a user (user), or some other way (other).

OpenAlex Levels

OpenAlex concept "levels" are such that broader concepts have smaller level values. You can think of the level as how deep in the concept hierarchy this concept resides. The "root" level is 0, and the narrowest level is 5.

Institution Metadata

All institution metadata (canonical/display name, abbreviation, parent/child relationships) were obtained from the adsabs/CanonicalAffiliations GitHub repository.

An Institution's Linked Collaborating Authors

These authors, sorted by display-name, have authored a work with someone (perhaps themselves) from this institution.

Concepts Associated with an Author

Author-concept associations are sourced from OpenAlex, which associates concepts with every Work in its corpus. These work-concept associations are then used in turn to associate a set of concepts with an Author, via that author's Works.

Author Funneling by Concepts

Funnel authors by an intersection of concept associations. The funneling will use a transitive closure for each selected concept -- specifically, its (narrower) descendants will be included. For example, funneling by Machine Learning will also find authors who are explicitly associated only with narrower concepts such as Artificial neural network.
